
## proyecto-vite

Pequeño ejemplo en React y Vite: un botón que alterna entre tema claro y oscuro usando `useState`.
Ademas se usa la componente card para ejercitar el uso de componentes, el cual contiene un boton dinamico que aumenta likes.

-----

## Tecnologías

- React
- Vite
- HTML
- CSS

-----

## Estructura principal

```
proyecto-vite/
├── src/
│   ├── App.jsx
│   ├── App.css
│   ├── Card.jsx
│   └── main.jsx
├── index.html
├── package.json
└── vite.config.js
```

-----

## Cómo ejecutar

Necesitas Node.js y npm (probado con Node >=16).

Instala dependencias y arranca el servidor de desarrollo:

```bash
npm install
npm run dev
```

Abre http://localhost:5173 en tu navegador.

-----

## Qué se practica aquí

- `useState` para controlar el tema (claro/oscuro) en `App.jsx`.
- Aplicar clases CSS dinámicas para cambiar estilos.
